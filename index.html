IXClocK -- Universal OS Dashboard

Panel de control visual con reloj, clima, música, IA, alarmas, notas, traductor, mapas y más. Todo funciona sin registro ni API key propia.

🕒 Reloj & Fecha

Haz clic en el reloj para cambiar entre modo Digital, Analógico y Flip. En Relojes puedes elegir estilo de fuente, color y si se muestran segundos. Activa el modo Romano para ver números romanos.

🖼️ Fondos de Pantalla

Más de 30 fondos con efecto Ken Burns (zoom suave). Cambia el fondo desde la cuadrícula en la pestaña Fondos. También puedes subir tu propia imagen con el botón 📁.

🌤️ Clima

El clima se actualiza automáticamente usando tu ubicación (Open-Meteo + OpenStreetMap). Acepta el permiso de ubicación cuando el navegador lo solicite. Muestra temperatura, humedad, viento, UV, probabilidad de lluvia y pronóstico horario.

⏰ Alarmas & Temporizador

Abre el panel con el icono ⏰ en la barra superior. Crea alarmas con hora, etiqueta y días de repetición. El temporizador hace cuenta regresiva y emite un sonido al terminar. Las alarmas se guardan localmente en tu navegador.

🧮 Calculadora

Cuatro modos: Básica, Científica (sin, cos, √, ^…), Programador (bin/oct/hex) y Conversión de unidades. Abre con el icono 🧮 de la barra superior.

📝 Notas

Panel de notas rápidas con soporte para múltiples notas, etiquetas de color y búsqueda. Las notas persisten entre sesiones gracias al almacenamiento local.

🤖 Asistente IA

Chat con IA usando Pollinations AI (gratuito, sin API key). Escribe tu pregunta y pulsa Enter o el botón Enviar. Historial de conversación visible en el panel flotante.

🎙️ Siri de Voz

Pulsa el icono del micrófono para activar el reconocimiento de voz (Web Speech API). Habla en español y el asistente responderá también en voz alta. Requiere Chrome o Safari con permiso de micrófono.

🌐 Traductor

Traduce texto entre más de 30 idiomas usando MyMemory API (gratuito). Selecciona idioma origen y destino, escribe o pega el texto y pulsa Traducir.

🎵 Radio & Música

Panel de radio con estaciones de todo el mundo. Haz clic en una estación para reproducir. Usa el control de volumen lateral. También hay sonidos ambiente (lluvia, bosque, café…) accesibles desde el panel de clima.

🗺️ Mapas

Mapa interactivo con Leaflet.js y OpenStreetMap (sin coste). Acepta la ubicación para centrar el mapa en tu posición. Puedes hacer zoom y arrastrar como en cualquier mapa.

🎹 GarageBand

Mini teclado virtual con Web Audio API. Elige instrumento (piano, sintetizador, guitarra…) y toca con el ratón o el teclado. Las notas se generan en tiempo real sin archivos de audio.

🧭 Brújula

Usa el sensor de orientación del dispositivo (funciona mejor en móviles). En escritorio muestra coordenadas GPS si el navegador lo permite.

☁️ iCloud (Backup local)

Guarda y restaura tu configuración (fondo, estilo de reloj, colores, notas, alarmas) en el almacenamiento local del navegador. No requiere cuenta ni conexión extra.

🎨 Personalización

Desde este panel: cambia idioma (ES/EN/FR/DE/PT/IT), modo oscuro/claro, formato 12h/24h, tamaño de fuente, visibilidad de cada sección y ajustes del efecto Liquid Glass (blur, saturación, brillo).
